# Monitoring

## Overview

Monitoring is a core part of this home lab. I use monitoring tools to gain visibility into system performance, service health, and resource usage.

## Monitoring Goals

- Track uptime
- View CPU, memory, and storage usage
- Detect performance issues
- Improve reliability
- Identify service failures quickly

## Tools

- Grafana
- Prometheus

## Outcome

This helped me build practical experience in observability, dashboard usage, and proactive system maintenance.
