# Services

## Overview

This lab includes self-hosted services for monitoring, automation, media management, and infrastructure support.

## Example Services

- Grafana
- Prometheus
- AdGuard Home
- Plex
- Jellyseerr
- Prowlarr
- Radarr
- qBittorrent
- n8n

## Responsibilities

- Deploying services in containers
- Updating and maintaining service configurations
- Troubleshooting service failures
- Monitoring uptime and performance
- Managing dependencies between services
